# hecl

The *H*askell *E*uler *C*ommon *L*ibrary for solving problems of [Project
Euler](https://projecteuler.net/)

## Installation

Only intended for local instances, clone the repository

```bash
git clone https://github.com/Entze/hecl.git
```

## Usage

```bash
stack ghci
```
